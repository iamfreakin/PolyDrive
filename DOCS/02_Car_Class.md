# 02. 클래스의 기초와 추상 클래스 (Car.h)

C++ 프로그래밍의 핵심인 **클래스(Class)**와 객체를 설계하는 방법을 알아봅니다.

## 1. 클래스란 무엇인가?
클래스는 현실 세계의 사물을 코드로 표현한 '설계도'입니다. 
- **멤버 변수(Data)**: 이름, 속도, 연비 등 차량의 상태.
- **멤버 함수(Behavior)**: 이동하기, 스펙 보여주기 등 차량의 동작.

## 2. 접근 제어자: `protected` vs `public`
`Car.h` 코드를 보면 변수들이 `protected`로 선언되어 있습니다.
- `private`: 나(해당 클래스)만 쓸 수 있음.
- `protected`: 나 그리고 **나를 상속받은 자식들**만 쓸 수 있음.
- `public`: 누구나 쓸 수 있음.

> **교육 팁**: 자식 클래스(`Bus` 등)가 부모의 변수(`name`, `baseSpeed`)에 직접 접근해야 하므로 `protected`를 사용했습니다.

## 3. 추상 클래스와 가상 함수
```cpp
virtual void ShowSpec() const { ... } // 가상 함수
virtual ~Car() {} // 가상 소멸자
```

### 왜 `virtual`인가?
- **다형성(Polymorphism)**을 위해 필요합니다. 부모 포인터(`Car*`)로 자식 객체를 가리킬 때, 실제 객체(`Bus`)의 함수가 호출되도록 약속하는 것입니다.
- **가상 소멸자**: 부모 포인터로 객체를 `delete`할 때 자식 클래스의 메모리까지 깨끗하게 지우기 위해 반드시 필요합니다.

### 추상 클래스(Abstract Class)란?
하나 이상의 순수 가상 함수를 가지거나, 객체를 직접 생성할 수 없는 '개념적'인 클래스입니다. 우리 게임에서 "그냥 차"라는 것은 존재하지 않고, 반드시 "버스"나 "트럭"이어야 하므로 `Car`는 설계도로만 존재합니다.
